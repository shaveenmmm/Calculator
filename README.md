## Features
 
- Basic arithmetic — addition, subtraction, multiplication, division
- Division by zero protection
- Clear button to reset
- Dark UI theme
- Keyboard-free — fully click-based interface
---
 
## Getting Started
 
### Prerequisites
 
- [Visual Studio 2022](https://visualstudio.microsoft.com/) with the **.NET Desktop Development** workload installed
- .NET 8.0 SDK
### Running the App
 
1. Clone the repository
   ```bash
   git clone https://github.com/shaveenmmm/Calculator
   ```
2. Open `Calculator.slnx` in Visual Studio
3. Press `Ctrl+Shift+B` to build
4. Press `F5` to run
---

## How It Works
 
- Clicking a number appends to the current display
- Clicking an operator stores the first number and waits for the second
- Clicking `=` computes and displays the result
- Clicking `C` resets everything back to `0`
